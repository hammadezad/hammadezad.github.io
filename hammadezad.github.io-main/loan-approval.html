Real-time Loan Approval System

Project | 2024 | Fintech ML

Summary: Built and deployed a machine learning model for automating loan approvals at a leading finance firm, reducing loan processing time by 40%.

Problem Statement

Manual loan approvals were slow and prone to errors. The client needed a reliable, scalable solution to predict approvals in real time.

What I Did

  • Cleaned and engineered features from 50,000+ historical loan applications
  • Used logistic regression and XGBoost to predict approval outcome and risk
  • Performed cross-validation, hyperparameter tuning, and feature selection
  • Deployed the model via a Flask REST API on AWS
  • Integrated the API with the companyâ€™s CRM for real-time predictions

Results & Impact

  • Processing time reduced from 2 days to 2 hours
  • Model achieved 89% accuracy (AUC: 0.92)
  • Enabled instant approvals for 70% of applications
